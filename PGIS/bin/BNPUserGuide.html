How to Use the Bits 'N' Pieces Parts Catalog

Welcome to the Bits 'N' Pieces User Guide for the Parts Catalog! The Parts Catalog allows you to store all kinds of products and retrieve them as you so desire. What's special about the Bits 'N' Pieces Parts Catalog is that there's no need to hire a DBA. Using complex, internal, magical powers, the Bits 'N' Pieces Parts Catalog takes your data and carefully organizes it into an XML file. You might be asking yourself the following question: "Why would anyone use XML to store parts information?" and the answer to that is "Because it's AWESOME!!" Think about it, why spend an eternity maintaining a database when you can simply store the same data in a file. Granted we wouldn't want to use this particular technique for exceptionally large amounts of data, but for personal usage or small sets of information, this works fantastic!

5. Expired Products:

Some products claim to be guaranteed for life while others have an expiration date. Good thing the Bits 'N' Pieces Parts Catalog comes equipped with its very own Expire-o-meter. As soon as you launch the application, the Expire-o-meter kicks into action and counts the products that have expired. If there are products that have exceeded their expiration date, the Bits 'N' Pieces Parts Catalog will tell you how many parts are no longer considered worthy of your needs.
